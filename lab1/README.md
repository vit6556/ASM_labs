# Лабораторная работа № 1 «Арифметическое выражение»

## Задание

Вычислить выражение:

$$ \frac{a^3 + b^3}{a^2 * c - b^2 * d + e} $$

## Параметры

- Данные: знаковые
- Размер переменной $a$: `16`
- Размер переменной $b$: `16`
- Размер переменной $c$: `32`
- Размер переменной $d$: `16`
- Размер переменной $e$: `32`
